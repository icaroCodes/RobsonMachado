# Robson Machado — Engenharia Civil & Consultoria Premium

![Tech Stack](https://img.shields.io/badge/Tech-React%20%2B%20Tailwind%20%2B%20Framer-brown?style=for-the-badge)

Portfolio imersivo e de alto padrão desenvolvido para **Robson Machado**, profissional especializado em Engenharia Civil, Gestão de Obras e Automação Residencial. O foco do projeto é transmitir autoridade técnica, precisão e sofisticação através de uma interface minimalista.

---

## 🎨 Design & Estética

O projeto segue uma paleta de cores curada para transmitir confiança e sobriedade:
- **Dark Green (#0E3B44):** Elegância técnica e solidez.
- **Brown (#B47045):** Detalhes orgânicos e refinamento.
- **Beige (#fdfcfb):** Fundo limpo e respiro visual.

### Destaques do UI/UX:
- **Adaptative Animations:** Sistema inteligente que reduz a complexidade das animações em dispositivos móveis para garantir 60 FPS constantes.
- **Minimalismo Apple:** Tipografia nítida, grandes espaços em branco e hierarquia visual clara.
- **Interatividade:** Galeria de projetos com modais imersivos e efeitos de transição fluídos.

---

## 🚀 Tecnologias Utilizadas

Este projeto utiliza o que há de mais moderno no ecossistema frontend:

| Tecnologia | Finalidade |
| :--- | :--- |
| **React 18** | Biblioteca base para a interface |
| **Tailwind CSS** | Estilização utilitária e responsividade extrema |
| **Framer Motion** | Orquestração de animações complexas e micro-interações |
| **Lucide React** | Conjunto de ícones minimalistas e consistentes |
| **Vite** | Ferramenta de build ultra-rápida |
| **Context API** | Gerenciamento de estado para o nível de animação (Performance Level) |

---

## 🛠️ Funcionalidades Principais

1.  **Home (Hero Section):** Primeiro impacto premium com CTAs claros para orçamento e portfólio.
2.  **Seção Sobre:** Trajetória profissional detalhada com foco em liderança técnica.
3.  **Habilidades (Diferenciais):** Lista interativa de abordagens técnicas que elevam o valor do projeto.
4.  **Galeria de Projetos:** Portfólio visual com filtragem adaptativa e visualização em alta resolução.
5.  **Benchmarking de Performance:** O sistema detecta se o usuário está no mobile ou desktop para servir a versão otimizada das animações.
6.  **Scroll Lock:** Experiência de visualização de projeto focada, bloqueando a rolagem da página principal durante o uso do modal.
